About

Here at the Summer Immersion Program at the Adobe San Francisco Office, we learn how to code through creating projects, including video games, art/animation, robotics, and websites. Each week has a different focus, and as we build up our knowledge of syntax of the different languages, we better our understanding of the basic components of code (e.g. loops, conditionals, lists, classes) and get to challenge our abilities. We have heard talks by several Adobe employees, including Emmy Huang, Divya Prabhakar, and Emilio Gambaretto, as well as others from various startups and companies, including Anna Killgore, Lucia Greco, and Albert Treat.

Scratch

Scratch is a Harvard-developed programming language. Block-based, it introduces basic programming "blocks" and components, including variables, lists, loops, and conditionals.

One of our first major projects in Scratch was Pig, where two players take turns rolling the dice. When a 1 is rolled, the turn passes to the next player. The first person to score 100 or more points wins. In this project, we used multiple sprites and many variables to keep track of the number rolled, the total for one turn, and the total score for each player. We used a random number generator to simulate the random rolling of the dice.

The last Scratch project was the Jukebox, which played a song from a specific set. We imported mp3 files to each sprite so that when the sprite (a button with the song title on it) is clicked, it plays the song. My partner Lily and I also created a feature where a user could create their own playlist. When they click the sprite "Create Playlist," and then select the songs in the order they want, it adds it to a playlist list. Once they click play, the songs will then play in that order. We also made a shuffle, where it would generate a random playlist and play it indefinitely.

Python

Python was our first foray into actual programming complete with syntax and functions. The first week, we focused on mastering functions and lists in Python, and the second week, we used Pygame, a library with more functions suited towards creating games.

Robotics

In our Robotics unit, we used a Parallax Uno bot connected to an Arduino board. Throughout the week, we experimented with circuits, Piezo elements, and touch sensors (whiskers) and used C++ to code in Arduino. Projects included the using LEDs to flash lights in time to a popular song, using a Piezo element to play a song using tones and frequencies, making the robot dance to a specific song, and using a touch sensor to enable the bot to self-navigate out of a three-walled room.
